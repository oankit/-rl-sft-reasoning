# Images Folder

Place your paper figures and images in this folder.

## Recommended Images:
- `overview.png` - Main overview figure showing your key results
- `method.png` - Diagram illustrating your methodology (optional)
- `results.png` - Visualization of your experimental results (optional)

## Image Guidelines:
- Use PNG format for diagrams and screenshots
- Use JPG format for photographs
- Optimize images for web (recommended max width: 1200px)
- Keep file sizes under 1MB when possible
- Use descriptive filenames
- Consider adding alt text in the HTML for accessibility

## Tools for Creating Figures:
- **Python**: Matplotlib, Seaborn, Plotly
- **Vector Graphics**: Adobe Illustrator, Inkscape
- **Diagram Tools**: draw.io, Figma, PowerPoint
- **Image Optimization**: TinyPNG, ImageOptim, Squoosh
